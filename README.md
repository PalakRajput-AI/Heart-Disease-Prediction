# ❤️ Heart Disease Prediction System

A Machine Learning based web application that predicts the possibility of heart disease using patient health data.

Built with **Python**, **Streamlit**, and **Scikit-learn**.

---

## 📖 About the Project

This project was developed as a student Machine Learning project to understand:

- Data preprocessing
- Feature scaling
- Machine Learning model training
- Model deployment using Streamlit
- Real-time prediction system

The app takes user health information and predicts whether the person has a **High Risk** or **Low Risk** of heart disease.

---

## 🚀 Features

✨ Simple and interactive UI  
✨ Real-time prediction  
✨ User input through sliders and dropdowns  
✨ Machine Learning model integration  
✨ Fast prediction results  
✨ Beginner-friendly project structure  

---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy

- Joblib

---

## 🧠 Machine Learning Model

This project uses the **K-Nearest Neighbors (KNN)** algorithm for prediction.

The model was trained on a heart disease dataset using:
- Data preprocessing
- Feature encoding
- Standard scaling
