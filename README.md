# ❤️ Heart Disease Prediction & Analysis

A student-level end-to-end Data Analytics and Machine Learning project developed using **Python, SQL, Power BI, and Streamlit**.

This project analyzes heart disease patient data, visualizes insights using Power BI, and predicts heart disease risk using a Machine Learning model.

---

# 📌 About the Project

The main goal of this project is to:

- Analyze heart disease data
- Find useful insights using SQL
- Create an interactive Power BI dashboard
- Build a Machine Learning prediction model
- Deploy the model using Streamlit

The application predicts whether a patient has a **High Risk** or **Low Risk** of heart disease based on medical details.
